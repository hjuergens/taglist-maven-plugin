Taglist Maven Plugin

The Taglist Maven Plugin generates a report on various tags found in the code, like @todo or //TODO tags.

Goals Overview

The Taglist Plugin has one goal which is meant to participate in the site life cycle:

  • taglist:taglist generates the taglist report.

Usage

Instructions on how to use the Taglist Plugin can be found in the usage guide.
